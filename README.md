# Scoop Bucket

Bucket for [Scoop](https://scoop.sh), the Windows command-line installer.

# Dev

```pwsh
scoop bucket add felamos https://github.com/felamos/scoopbucket
scoop install felamos/<manifestname>
scoop update felamos
```
